Tykode offers a beautiful theme system that allows you to customize your viewing experience with light and dark modes.

## Theme Toggle

The theme toggle is easily accessible throughout the platform, allowing you to switch between light and dark modes at any time.

### Accessing Theme Settings

You can change your theme preference in multiple ways:

**Desktop Navigation**
- The theme toggle button is located in the top-right section of the header
- Click the sun/moon icon to instantly switch themes

**Mobile Navigation**
- Open the mobile menu by tapping the menu icon
- Scroll to the bottom of the menu
- Find the "Theme" section with the toggle control

<Info>
  Your theme preference is automatically saved and will persist across sessions and devices when you're logged in.
</Info>

## How to Change Theme

<Steps>
  <Step title="Locate the theme toggle">
    Find the theme toggle button in the header (desktop) or mobile menu. It displays a sun icon in light mode and a moon icon in dark mode.
  </Step>
  <Step title="Click to switch">
    Click or tap the theme toggle button. The theme will change instantly with a smooth transition.
  </Step>
  <Step title="Enjoy your new theme">
    The entire interface updates immediately, and your preference is saved automatically.
  </Step>
</Steps>

## Theme Options

### Light Mode

Light mode provides a bright, clean interface ideal for:

- Well-lit environments
- Daytime use
- Users who prefer high contrast
- Reduced eye strain in bright conditions

**Light Mode Features:**
- White and light gray backgrounds
- Dark text for optimal readability
- Subtle shadows and borders
- Tykode's signature green accent color (#22C55E)

### Dark Mode

Dark mode offers a sophisticated, eye-friendly experience perfect for:

- Low-light environments
- Evening and night use
- Reduced eye strain in dark conditions
- OLED screen battery savings
- Professional and modern aesthetic

**Dark Mode Features:**
- Deep black and dark gray backgrounds
- Light text with reduced brightness
- Enhanced green accents that pop
- Reduced blue light emission

<Note>
  The theme toggle uses smooth CSS transitions to prevent jarring changes when switching between modes.
</Note>

## System Preference Detection

Tykode respects your system-level theme preferences:

- **First Visit**: The platform detects your operating system's theme setting
- **Auto-Sync**: If you haven't set a preference, the theme follows your system settings
- **Manual Override**: Once you manually select a theme, your choice takes precedence

### How It Works

The theme system uses `next-themes` to:

1. Detect your system preference on first load
2. Apply the appropriate theme automatically
3. Store your manual selection in local storage
4. Sync preferences across browser tabs

## Theme Persistence

Your theme preference is saved using multiple methods:

**Local Storage**
- Theme choice is stored in your browser
- Persists between sessions
- Loads instantly on page refresh

**User Preferences** (when logged in)
- Synced to your account
- Consistent across all your devices
- Backed up with your profile data

<Info>
  If you clear your browser data, the theme will reset to your system preference or light mode by default.
</Info>

## Accessibility Considerations

The theme system is designed with accessibility in mind:

**Contrast Ratios**
- All text meets WCAG AA standards (4.5:1 minimum)
- Important elements exceed AAA standards (7:1+)
- Links and interactive elements are clearly distinguishable

**Color Independence**
- Information is never conveyed by color alone
- Icons and labels supplement color-coded elements
- Pattern and position provide additional context

**Reduced Motion**
- Theme transitions respect `prefers-reduced-motion` settings
- Animations are minimal and functional
- Instant theme changes for users who prefer no motion

<Note>
  Users with motion sensitivity can disable theme transition animations through their operating system's accessibility settings.
</Note>

## Technical Implementation

For developers interested in the technical details:

### Technology Stack

- **next-themes**: Theme management and persistence
- **Tailwind CSS**: Theme-aware styling with dark: prefix
- **CSS Variables**: Dynamic color tokens
- **React Context**: Theme state management

### Theme Toggle Component

The theme toggle is implemented as a simple, accessible button:

```tsx
<Button variant="outline" size="icon" onClick={toggleTheme}>
  <Sun className="h-[1.2rem] w-[1.2rem] rotate-0 scale-100 transition-all dark:-rotate-90 dark:scale-0" />
  <Moon className="absolute h-[1.2rem] w-[1.2rem] rotate-90 scale-0 transition-all dark:rotate-0 dark:scale-100" />
  <span className="sr-only">Toggle theme</span>
</Button>
```

The component:
- Uses semantic icons (sun for light, moon for dark)
- Includes screen reader text for accessibility
- Animates smoothly between states
- Works with keyboard navigation

## Troubleshooting

### Theme Not Switching

If the theme toggle doesn't work:

1. **Check JavaScript**: Ensure JavaScript is enabled in your browser
2. **Clear Cache**: Try clearing your browser cache and cookies
3. **Update Browser**: Make sure you're using a modern browser version
4. **Check Extensions**: Disable browser extensions that might interfere

### Theme Not Persisting

If your theme resets:

1. **Browser Storage**: Enable local storage in browser settings
2. **Private Browsing**: Note that private/incognito mode may not save preferences
3. **Login**: Sign in to sync preferences across devices

### Visual Glitches

If you see visual issues when switching themes:

1. **Refresh Page**: Try a hard refresh (Ctrl+F5 or Cmd+Shift+R)
2. **Update Browser**: Ensure you're using the latest browser version
3. **Report Issue**: Contact support with browser details and screenshots

<Info>
  The theme system is built on modern web standards and works best in up-to-date browsers like Chrome, Firefox, Safari, and Edge.
</Info>
